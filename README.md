# 写真共有アプリ(Photo Share)
Video Demo: <https://youtu.be/KSshwt_le4A>

## 何をするアプリなのか
結婚式や運動会など同じ目的で集まったが、互いには面識がなく写真を共有したいと思っても「SNSのアカウントなどを共有するのが面倒」
そんな問題を解決するために作成したアプリです。

## 開発に用いた言語およびフレームワーク
* HTML/CSS
* Bootstrap
* Python
* SQLite
* Django

## 各機能の説明
* ホーム画面の「はじめる」から写真フォルダーを作製するオーナー用のアカウントを作成することができます。
* 写真「写真フォルダーのオーナーへログイン」からオーナー用のページにログインできます。
* オーナー用のページでは写真フォルダーの作製ができ、その際に写真フォルダー名とパスワードを決めます。また写真フォルダーの削除もできます。
* 「写真フォルダーへログイン」では写真フォルダーにアクセスしたいゲストがオーナーが決めた写真フォルダー名とパスワードで写真フォルダーへログインができます。
* 写真フォルダーへログインした後の写真フォルダーのページでは写真のアップロード、削除、ダウンロードが可能です。

## こだわった点
* 「SNSのアカウントなどを共有するのが面倒」という個人情報を共有するのに抵抗がある人を想定しているので、写真フォルダーを作製するオーナー以外はアカウント作成不要で使える点
* オーナーのアカウント作成時に簡単すぎるパスワードには制限をかけセキュリティーを意識した開発が行えた点
* 写真フォルダーの画像を表示するページでBootstrapのカードを用いて削除、ダウンロードを一つのセットにした点

## 改善点
* 大枠を決めるのに時間を要してしまい最低限必要な機能は実装できたが、画像判定などの追加の機能実装の実装にいたらなぁった。
* ホームのページ、バナー、写真フォルダーの画像を表示するページではBootstrapを用いて視覚的にモダンなフロントエンドの実装ができたが、時間が足らず細部に至るまで画面詳細図に従った実装はできなかった。
